## Structural Notes (Consistent with v1.0)
- Identity glyphs: vertical symmetry  
- Emotion glyphs: curved forms  
- Logic glyphs: angular geometry  
- Action glyphs: forward‑leaning strokes  
- Relation glyphs: interlocking shapes  
- Security glyphs: closed forms  

Compound glyphs follow:  
- Horizontal linking = relation  
- Vertical stacking = modification  
- Overlay = security  
- Encirclement = group semantics  
- Split strokes = branching logic
