## Structural Design Rules (SVG Guidelines)

### Shape Rules
- Identity: vertical symmetry  
- Emotion: curved forms  
- Logic: angular geometry  
- Action: forward‑leaning strokes  
- Relation: interlocking shapes  
- Security: closed forms  

### Stroke Rules
- Base glyphs: uniform stroke  
- Compound glyphs: base strokes + thinner linking strokes  

### Visual Grammar
- Horizontal linking = relation  
- Vertical stacking = modification  
- Overlay = security  
- Encirclement = group semantics  
- Split strokes = branching logic
